Система спроектирована как Single-User (локальное приложение), поэтому выделяется только одна функциональная роль.

**Пользователь (Владелец системы)**

- **Доступы:** Полный и неограниченный доступ ко всем функциям UI и локальной БД (JSON-файлам)  
- **Ключевые функции:**
    - Базовая настройка (ввод API-ключа Claude, установка лимитов токенов в 10k/15k).
    - Управление агентами (задание количества, редактирование system prompt для Оптимиста/Пессимиста/Реалиста).
    - Контроль жизненного цикла итераций (старт, пауза, `forced_close`, вызов Синтезатора).
    - Работа с архивом (просмотр, экспорт и очистка логов сессий).
