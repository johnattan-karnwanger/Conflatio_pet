#### Бизнес-правила

1. Перед запуском дискуссии должен быть активен хотя бы один агент дискуссии.
2. Синтезатор может быть запущен только после получения хотя бы одного ответа от агента дискуссии.
3. Итог, сформированный до ответа всех активных агентов дискуссии, считается потенциально неполным.
4. При ручной остановке до завершения всех активных агентов система должна явно предупредить пользователя о возможной неполноте результата.
5. Одна пользовательская сессия должна храниться как единый логический журнал дискуссии.
