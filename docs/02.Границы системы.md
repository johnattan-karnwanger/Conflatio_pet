#### Границы системы

| Внутри границы системы | Снаружи границы системы |
|---|---|
| Desktop UI | Пользователь |
| Backend приложения | Claude API / API LLM-провайдера |
| Локальное хранилище сессий (JSON-файлы) | Файловая система ОС |

#### Пояснение

Пользователь взаимодействует с системой через Desktop UI.  
Backend приложения управляет логикой итераций, формированием запросов к моделям, обработкой ответов и сохранением сессий.  
Внешними зависимостями по отношению к системе являются API LLM-провайдера и среда исполнения ОС.
