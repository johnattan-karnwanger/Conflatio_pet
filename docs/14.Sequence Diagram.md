#### 1 Сценарий "Установка API"

``` mermaid
sequenceDiagram
    Пользователь->>Система:Зашел в настройки
    Система-->>Пользователь: Вывелось окно с API
    Пользователь->>Система: Ввел API
    Система->>LLM API: Обращение к API
    alt Успех
        LLM API-->>Система: Вернул ответ
        Система-->>Система: Сохранение и шифрование API
    else Ошибка 401
        LLM API-->>Система: 401
        Система-->>Пользователь: Неверный API-ключ
    else Ошибка 402/429
        LLM API-->>Система: 402/429
        Система-->>Пользователь: Кончился билинг
    else Недоступен
        LLM API-->>Система: Недоступен
        Система-->>Пользователь: Провайдер не отвечает
    end
```
#### 3 Сценарий "Дискуссия с LLM"

```mermaid
sequenceDiagram
    Пользователь->>Система: Отправляет запрос
    Система->>LLM API: Обращение к API
    alt Успех
        LLM API-->>Система: Ответ
        Система->>Оптимист: Вызов роли
        Оптимист-->>Система: Ответ
        Система->>Пессимист: Вызов роли
        Пессимист-->>Система: Ответ
        Система->>Реалист: Вызов роли
        Реалист-->>Система: Ответ
        Система-->>Пользователь: Пауза — выбор действия
        alt Завершить + Синтезатор
            Пользователь->>Система: Завершить + Синтезатор
            Система->>Синтезатор: Вызов роли
            Синтезатор-->>Система: Финальный ответ
            Система-->>Пользователь: Финальный ответ
        else Завершить и остановить
            Пользователь->>Система: Завершить и остановить
            Система-->>Пользователь: Итерация Stopped
        end
    else Ошибка 401
        LLM API-->>Система: 401
        Система-->>Пользователь: Неверный API-ключ
    else Ошибка 402/429
        LLM API-->>Система: 402/429
        Система-->>Пользователь: Кончился биллинг
    else Недоступен
        LLM API-->>Система: Недоступен
        Система-->>Пользователь: Провайдер не отвечает
    else Лимит заполнен
		LLM API-->>Система: 400
		Система-->>Пользователь: Контекст переполнен, очистите историю
    end
```
