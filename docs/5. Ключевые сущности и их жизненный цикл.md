
#### 5.1. Основные сущности системы

| **Сущность**        | **Описание**                                   | **Состав данных (JSON)**                                               |
| ------------------- | ---------------------------------------------- | ---------------------------------------------------------------------- |
| **Чат (Сессия)**    | Корневой объект. Хранится как один файл на ПК. | `session_id`, `created_at`, `config` (лимиты 10k/15k), `iterations[]`. |
| **Итерация**        | Контейнер для одного цикла дискуссии.          | `iteration_id`, `status` (Enum), `messages[]`.                         |
| **Ответ (Message)** | Единица диалога (запрос или ответ агента).     | `message_id`, `role`, `content`, `timestamp`, `tokens_used`.           |

---

#### 5.2. Жизненный цикл Итерации (Статусы)

| **Статус**      | **Описание**                                                        | **Триггер перехода**            |
| --------------- | ------------------------------------------------------------------- | ------------------------------- |
| **In_Process**  | Идет активная генерация ответов Оптимиста, Пессимиста или Реалиста. | Отправка запроса в API LLM.     |
| **Stopped**     | Пауза. Сработал `forced_close`.                                     | Ручная остановка пользователем. |
| **Synthesized** | Финальный агент (Синтезатор) сформировал итоговый ответ.            | Успешная запись ответа от LLM.  |
