
| **Термин**                                 | **Определение**                                                                                                                | **Синонимы / Связанные понятия** |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------ | -------------------------------- |
| **Conflatio**                              | Проектируемая система мультиагентной дискуссии на базе LLM.                                                                    | Целевая система, Приложение      |
| **Агент (Agent)**                          | Экземпляр LLM, которому назначен системный промпт (роль) для выполнения специфичной задачи в рамках итерации.                  | Модель, Роль                     |
| **Оптимист / Пессимист / Реалист**         | Базовые роли Агентов в пайплайне генерации, выполняющие функции генерации идей, критики и валидации соответственно.            | Роли                             |
| **Синтезатор**                             | Финальный Агент (Claude Haiku), формирующий итоговый вывод на основе дискуссии предыдущих ролей.                               | Сумматор                         |
| **Итерация (Iteration)**                   | Один полный цикл последовательного взаимодействия Агентов.                                                                     | Проход, Цикл                     |
| **Статусы итерации**                       | Состояния жизненного цикла итерации в БД:                                                                                      | Жизненный цикл итерации          |
| **Лог дискуссии (Log)**                    | Структурированная история сообщений (context window) в формате JSON, передаваемая Агентам.                                     | Контекст, История чата           |
| **Системный промпт (System prompt)**       | Базовая инструкция, задающая контекст, ограничения и модель поведения конкретного Агента.                                      | System message, Ролевой промпт   |
| **Авто-саммаризация (Auto-summarization)** | Фоновый процесс сжатия старых сообщений лога в краткую выжимку при достижении лимита (по умолчанию 10 000 токенов).            | Сжатие контекста                 |
| **Скользящее окно (Sliding window)**       | Механизм управления памятью, передающий в API только срез из последних N токенов (по умолчанию 15 000), отсекая старые данные. | Context window management        |
| **LLM**                                    | Large Language Model — большая языковая модель (в данном проекте семейство Claude 4.6), выступающая ядром для Агентов.         | Языковая модель                  |
| **Чат**                                    | Корневой объект системы, представляющий собой отдельный проект/диалог, сохраненный в локальном JSON-файле.                     | Диалог, сессия                   |
| **Ответ (Message)**                        | Минимальная единица данных в логе: текст, сгенерированный Агентом, или запрос Пользователя.                                    | Сообщение, Реплика               |
